## ‚ú® Core Features (Live Now)

### 1. üì∏ Advanced Image Forensics
TruthLens looks past the pixels to find the flaws in the math.
* **The "Logic Lock":** Automatically validates document metadata (Dates, Transaction IDs) against real-world logic (Julian Dates, Calendar consistency).
* **Texture Physics:** Scans for the absence of natural sensor noise (ISO grain). If an image is "too smooth," it‚Äôs flagged.
* **Vector Hallucinations:** Detects "wobbly" lines and asymmetrical logos that generative models struggle to render perfectly.

### 2. üé• Video Deepfake Detection
Static analysis isn't enough. TruthLens analyzes **Temporal Consistency** across frames.
* **Glitch Detection:** Scans for micro-artifacts where a deepfake face "detaches" or morphs unnaturally during rapid movement.
* **Lighting Consistency:** Verifies if the lighting on the subject matches the environment across the entire timeline.
* **Lip-Sync Fidelity:** Checks for subtle desynchronization between audio phonemes and visual mouth shapes.

### 3. ‚è≥ Temporal Grounding (Anti-Hallucination)
AI models often live in a timeless void. We solved this.
* **Dynamic Anchoring:** We inject real-time `{{CURRENT_DATE}}` data into the system prompt.
* **Benefit:** Prevents the AI from hallucinating that valid 2025 dates are "in the future" or accepting expired documents as new.
